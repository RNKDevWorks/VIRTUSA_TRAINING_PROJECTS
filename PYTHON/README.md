<h1 align="center"> Smart Monthly Expense Tracker </h1>

* The Smart Monthly Expense Tracker is a Python-based console application that helps users record, manage, and analyze their daily expenses.
* It stores expense data in a CSV file and provides summaries, category-wise analysis, and budget insights with visual charts.
