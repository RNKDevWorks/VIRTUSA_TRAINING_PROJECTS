### SHOPPING MANAGEMENT SYSTEM
A console-based Shopping Management System built using Core Java that simulates real-world shopping operations with Admin and Customer roles.

--- 

### Features
#### Admin
* Sign up & Login  & Add new products
![Login](https://github.com/RNKDevWorks/VIRTUSA_TRAINING_PROJECTS/blob/main/JAVA/OUTPUTS/5.jpeg)
* View all products
![Admin Menu](https://github.com/RNKDevWorks/VIRTUSA_TRAINING_PROJECTS/blob/main/JAVA/OUTPUTS/6.jpeg)
#### Customer
* SignUp
![signUp](https://github.com/RNKDevWorks/VIRTUSA_TRAINING_PROJECTS/blob/main/JAVA/OUTPUTS/j1.jpeg)
* LOG IN & View available products & Add products to cart
![VieW&Add](https://github.com/RNKDevWorks/VIRTUSA_TRAINING_PROJECTS/blob/main/JAVA/OUTPUTS/2.jpeg)
* Checkout and generate bill
![Billing](https://github.com/RNKDevWorks/VIRTUSA_TRAINING_PROJECTS/blob/main/JAVA/OUTPUTS/3.jpeg)
![Checkout](https://github.com/RNKDevWorks/VIRTUSA_TRAINING_PROJECTS/blob/main/JAVA/OUTPUTS/4.jpeg)

---

### Technologies Used
* Java (Core Java)
* OOP Concepts (Abstraction, Inheritance, Polymorphism, Encapsulation)
* Java Collections (ArrayList)
* Scanner (User Input)
